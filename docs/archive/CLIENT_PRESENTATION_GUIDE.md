# 客户演示材料指南

## 演示材料位置

所有演示材料已生成在 `demo_presentation/` 目录：

1. **01_system_overview.jpg** - 系统概览
   - 系统名称和主要功能
   - 技术栈说明
   - 适用于：项目介绍、功能概览

2. **02_architecture.jpg** - 系统架构图
   - 系统组件和流程
   - 数据流向
   - 适用于：技术方案说明

3. **03_detection_example.jpg** - 检测示例
   - 模拟检测结果展示
   - 检测框和标签
   - 适用于：功能演示

4. **04_configuration.jpg** - 配置摘要
   - 系统配置参数
   - 硬件和软件配置
   - 适用于：技术细节说明

5. **05_performance.jpg** - 性能指标
   - 系统性能数据
   - 准确率和速度指标
   - 适用于：性能展示

## 使用建议

### 演示顺序
1. 系统概览（介绍项目）
2. 系统架构（技术方案）
3. 检测示例（功能演示）
4. 性能指标（性能说明）
5. 配置摘要（技术细节）

### 配合文档
- `docs/PROJECT_DOCUMENTATION.md` - 完整项目文档
- `docs/field_test_optimization_summary.md` - 现场测试优化总结
- `README.md` - 项目说明

## 其他可用材料

### 配置文件
- `config.yaml` - 系统配置文件（可展示配置灵活性）

### 文档
- `docs/` 目录下的各种技术文档

### 代码结构
- 项目代码结构清晰，模块化设计
- 可以展示代码质量和架构设计

## 注意事项

1. 这些是演示材料，不是实际运行结果
2. 实际系统已在现场测试，功能正常
3. 视频文件因录制中断而损坏，无法展示
4. 建议在有机会时进行现场演示或重新录制

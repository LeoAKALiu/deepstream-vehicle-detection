# 硬编码参数列表

## 📋 当前硬编码的参数

### 1. 网络配置
- `cassia-ip`: `192.168.1.2` (命令行参数默认值)
- `camera-id`: `camera_01` (命令行参数默认值)

### 2. 检测阈值
- `conf_threshold`: `0.5` (检测置信度阈值，在`postprocess`方法中)
- `iou_threshold`: `0.4` (NMS的IoU阈值，在`postprocess`方法中)

### 3. 跟踪参数
- `iou_threshold`: `0.3` (跟踪匹配的IoU阈值，在`VehicleTracker.__init__`中)
- `max_age`: `30` (跟踪消失的最大帧数，在`VehicleTracker.__init__`中)

### 4. 文件路径
- `beacon_whitelist.yaml`: 相对路径 `os.path.join(os.path.dirname(__file__), 'beacon_whitelist.yaml')`
- `models/custom_yolo.engine`: 默认模型路径
- 字体路径: `/usr/share/fonts/opentype/noto/NotoSansCJK-Regular.ttc`
- 共享缓冲区: `/tmp/orbbec_shared_frame.npy`, `/tmp/orbbec_shared_depth.npy`

### 5. 显示和性能参数
- `fps_frame_count >= 10`: FPS计算间隔（每10帧计算一次）
- `cv2.waitKey(1)`: 显示刷新间隔（1ms）
- `time.sleep(0.01)`: 帧获取失败时的等待时间
- 字体大小: `24` (大字体), `18` (小字体)
- 标签位置偏移: `y1 - 30` (标签在bbox上方30像素)

### 6. 深度测量参数
- 深度有效范围: `0.1-10m` (在`get_depth_region_stats`中，100-10000mm)
- 深度测量方法: `'median'` (默认使用中位数)

### 7. 车牌识别参数
- ROI最小尺寸: `20x20` (在`check_civilian_vehicle`中)

### 8. 颜色配置
- 工程车辆: `(0, 140, 255)` (橙色)
- 社会车辆: `(0, 255, 0)` (绿色)
- 未备案: `(0, 0, 255)` (红色)

### 9. 信标过滤参数（在`beacon_whitelist.yaml`中，但可能需要主程序配置）
- RSSI阈值（在beacon_filter.py中，但可能需要全局配置）
- 距离容差（在beacon_filter.py中）

### 10. 日志和输出（待实现）
- 日志文件路径: 未实现
- 报警记录路径: 未实现

## 🎯 建议配置化的参数

### 高优先级（建议配置化）
1. **网络配置**
   - `cassia-ip`: 蓝牙路由器IP地址
   - `camera-id`: 摄像头ID

2. **检测阈值**
   - `conf_threshold`: 检测置信度阈值
   - `iou_threshold`: NMS的IoU阈值

3. **跟踪参数**
   - `track_iou_threshold`: 跟踪匹配的IoU阈值
   - `max_age`: 跟踪消失的最大帧数

4. **文件路径**
   - `model_path`: 模型文件路径
   - `beacon_whitelist_path`: 信标白名单路径
   - `log_path`: 日志文件路径
   - `alert_log_path`: 报警记录路径

### 中优先级（可选配置化）
5. **显示参数**
   - `fps_update_interval`: FPS更新间隔（帧数）
   - `font_size_large`: 大字体大小
   - `font_size_small`: 小字体大小

6. **性能参数**
   - `frame_wait_time`: 帧获取失败等待时间
   - `display_wait_key`: 显示刷新间隔

7. **深度测量参数**
   - `depth_min_range`: 最小深度范围（米）
   - `depth_max_range`: 最大深度范围（米）
   - `depth_method`: 深度测量方法（median/mean/min）

8. **车牌识别参数**
   - `lpr_min_roi_size`: ROI最小尺寸

### 低优先级（可保持硬编码）
9. **颜色配置**（通常不需要修改）
10. **字体路径**（系统字体，通常不需要修改）
